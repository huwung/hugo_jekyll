
# Introduction
首先，我是hugo，并不是程序猿。

其实，两者是不一样的。

尽管，他们的工作有相似的部分。

交叉学科是未来发展最快的领域。
希望朋友们能在这里获得新的灵感，继而把这些ideas实现。
